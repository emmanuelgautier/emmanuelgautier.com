## Markdown

```shell:rename-multiple-fileextensions-markdown.sh
for x in **/*.md; do mv "$x" "${x%.md}.mdx"; done
```

## Typescript

```shell:rename-multiple-fileextensions-typescript.sh
for x in **/*.js; do mv "$x" "${x%.md}.ts"; done
```
